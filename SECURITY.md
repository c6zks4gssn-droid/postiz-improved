## Reporting a Vulnerability
- Email: passiveassets@proton.me
- Or open a private security advisory on GitHub

## Security Features (Bonanza Labs Improvements)
- Bonanza Auth: API key + JWT authentication for agents
- Agent Wallet: Policy-based spending limits
- Trivy container scanning in CI
- SBOM generation on every push
- Non-root Docker containers
